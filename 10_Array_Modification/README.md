# Array Modification in NumPy

## 📖 Overview

This folder demonstrates different techniques to modify NumPy arrays.

---

## Programs

### 1. append_elements.py

Append one or multiple elements to an array.

Function Used

- np.append()

---

### 2. concatenate_arrays.py

Merge two arrays into one.

Function Used

- np.concatenate()

---

### 3. insert_element_1d.py

Insert an element at a specific position in a one-dimensional array.

Function Used

- np.insert()

---

### 4. insert_element_2d.py

Insert a row into a two-dimensional array.

Function Used

- np.insert(axis=0)

---

## Concepts Covered

- Append
- Concatenate
- Insert
- 1D Arrays
- 2D Arrays
