# Array Operations in NumPy

## 📖 Overview

This folder demonstrates common NumPy array operations such as aggregation functions, arithmetic operations, and data type conversion.

---

## 📂 Programs

### 1. aggregation_functions.py

Demonstrates common aggregation functions.

Functions Covered

- np.sum()
- np.mean()
- np.min()
- np.max()
- np.std()
- np.var()

---

### 2. arithmetic_operations.py

Demonstrates element-wise arithmetic operations.

Operations Covered

- Addition
- Subtraction
- Multiplication
- Division
- Power

---

### 3. type_conversion.py

Demonstrates data type conversion using `astype()`.

---

## 📚 Concepts Covered

- Aggregation Functions
- Arithmetic Operations
- Type Conversion
- Vectorized Computation
