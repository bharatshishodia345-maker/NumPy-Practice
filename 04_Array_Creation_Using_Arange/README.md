# Array Creation Using np.arange()

## Overview

This program demonstrates how to create sequences of numbers using NumPy's `arange()` function.

The `np.arange()` function generates evenly spaced values within a specified range.

---

## Syntax

```python
np.arange(start, stop, step)
```

### Parameters

- **start** → Starting value (inclusive)
- **stop** → Ending value (exclusive)
- **step** → Increment between values

---

## Example

```python
np.arange(1, 10, 1)
```

Output

```
[1 2 3 4 5 6 7 8 9]
```
