# Two-Dimensional Array in NumPy

## Overview

This program demonstrates how to create and display a two-dimensional (2D) NumPy array.

A 2D array represents data in rows and columns, similar to a matrix or spreadsheet.

---

## Concepts Covered

- NumPy
- 2D Arrays
- Matrix Representation
- Array Creation
- Printing Arrays

---

## Requirements

- Python 3.x
- NumPy

Install NumPy

```bash
pip install numpy
```

---

## Code Example

```python
import numpy as np

array_2d = np.array([
    [4, 5, 6],
    [7, 8, 9],
    [1, 2, 3]
])

print(array_2d)
```

---

## Output

```
[[4 5 6]
 [7 8 9]
 [1 2 3]]
```

---
### Example 2

```python
import numpy as np

array_2d = np.array([
    [55, 45, 63],
    [8, 6, 4]
])

print(array_2d)
```

Output

```
[[55 45 63]
 [ 8  6  4]]
```
