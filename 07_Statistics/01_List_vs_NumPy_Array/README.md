# List vs NumPy Array

## Overview

This program demonstrates the basic difference between a Python List and a NumPy Array.

Both data structures can store multiple values, but NumPy arrays are optimized for numerical computations and scientific computing.

---

## Concepts Covered

- Python List
- NumPy Array
- Importing NumPy
- Creating Arrays
- Printing Arrays

---

## Requirements

- Python 3.x
- NumPy

Install NumPy

```bash
pip install numpy
```

---

## Output

Python List

```
[3, 4, 5, 6, 9, 8, 2, 3, 5]
```

NumPy Array

```
[3 4 5 6 9 8 2 3 5]
```
