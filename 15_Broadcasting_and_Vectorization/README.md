# NumPy Broadcasting and Vectorization

## 📖 Overview

This module demonstrates the concepts of **Broadcasting** and **Vectorization** in NumPy. These techniques allow mathematical operations on arrays without using explicit loops, resulting in cleaner, faster, and more efficient code.

---

## 📂 Programs Included

### 1. Matrix and Vector Broadcasting
- Add a 2D matrix with a 1D vector
- Demonstrates NumPy Broadcasting

**Function Used**
- `+`

---

### 2. Discount Calculator
- Apply a percentage discount to product prices
- Uses broadcasting with scalar values

**Concept**
- Broadcasting

---

### 3. Scalar Multiplication
- Multiply every element of an array by a constant value

**Concept**
- Vectorization

---

### 4. Vector Addition
- Add two NumPy arrays element-wise

**Concept**
- Vectorized Addition

---

### 5. Array Multiplication
- Multiply every element in an array by a scalar

**Concept**
- Vectorized Multiplication
