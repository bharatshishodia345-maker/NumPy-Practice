# Array Reshaping

## Overview

This folder demonstrates how to change the shape of NumPy arrays.

---

## Programs

### 1. reshape_array.py

Convert a 1D array into a 2D array using `reshape()`.

---

### 2. flatten_and_ravel.py

Convert a multidimensional array into a 1D array using:

- ravel()
- flatten()

---

## Concepts Covered

- reshape()
- ravel()
- flatten()
