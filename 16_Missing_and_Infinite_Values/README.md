# NumPy Missing and Infinite Values

## 📖 Overview

This module demonstrates how to detect and handle missing (`NaN`) and infinite (`Inf`) values in NumPy arrays.

These techniques are commonly used during data preprocessing in Data Science and Machine Learning.

---

## 📂 Programs Included

### 1. Detect NaN Values
- Detect missing values
- Use `np.isnan()`

### 2. Fill NaN Values
- Replace missing values with default or custom values
- Use `np.nan_to_num()`

### 3. Detect Infinite Values
- Detect positive and negative infinity
- Use `np.isinf()`

### 4. Replace Infinite Values
- Replace positive and negative infinity
- Use `np.nan_to_num()`
